# Student Result Management System

A Python Tkinter-based Student Result Management System with SQLite database.

# Features
- User Login / Logout
- Manage Courses
- Manage Students
- Manage Results
- View and Search Results
- SQLite Database Integration
- Modern Tkinter UI
